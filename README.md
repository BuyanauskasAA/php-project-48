## Setup

``` bash
make install
```

## Usage

```
gendiff [--format <fmt>] <firstFile> <secondFile>
```

## Description

Gendiff generates difference between two files.  
Supports files in JSON and YAML formats.  
Gendiff includes three output formats:  

* stylish (default format)
* slain
* json
